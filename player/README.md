# Player - 文本播放器

一个基于Web的文本播放器，通过服务端API获取书籍内容并进行分段播放。

## 功能特性

1. **播放窗口**
   - 占满屏幕宽度
   - 固定1/2屏幕高度
   - 显示当前页的句子

2. **书籍章节选择**
   - 调用 `/books/list` 获取书名列表
   - 调用 `/chapter/{book_name}` 获取章节列表
   - 支持章节切换（上一章/下一章）

3. **流式加载**
   - 调用 `/chapter/{book_name}/{chapter_name}` 获取章节内容
   - 支持按位置和最小长度获取内容
   - 自动预加载下一段内容

4. **分句显示**
   - 调用 `/split` 接口进行分句
   - 大字体、加粗显示
   - 类似字幕效果
   - 淡入动画效果

5. **自动播放**
   - 支持三种播放速度：慢(3s)、中(2s)、快(1s)
   - 播放/暂停控制
   - 上一页/下一页导航
   - 跳转到开头/结尾

## 使用方法

### 1. 启动分句服务

首先在另一个终端启动分句服务：

```bash
cd ../server
python service_manager.py start
```

### 2. 打开播放器

直接在浏览器中打开 `player.html` 文件：

```
直接双击 player.html 文件
或在浏览器中打开: file:///path/to/player/player.html
```

### 3. 操作步骤

1. 从下拉菜单选择书籍
2. 选择要阅读的章节
3. 点击「加载」按钮加载章节内容
4. 点击绿色播放按钮开始播放
5. 使用速度按钮调整播放速度
6. 使用导航按钮手动翻页
7. 使用「上一章/下一章」切换章节

## API依赖

播放器依赖以下服务端API：

| 接口 | 用途 |
|------|------|
| GET `/books/list` | 获取书名列表 |
| GET `/chapter/{book_name}` | 获取书籍的章节列表 |
| GET `/chapter/{book_name}/{chapter_name}?position=X&min_size=Y` | 获取章节内容 |
| POST `/split` | 对文本进行分句 |

## API配置

默认连接到 `http://localhost:8000` 的分句服务。

如需修改API地址，编辑 `player.html` 中的：

```javascript
const API_URL = 'http://localhost:8000';
```

## 自定义配置

### 修改默认获取的内容大小

编辑 `player.html`：

```javascript
const DEFAULT_MIN_SIZE = 500; // 每次获取的最小字符数
```

### 修改默认播放速度

编辑 `player.html`：

```javascript
let playSpeed = 2000; // 毫秒，2000 = 2秒
```

### 修改每页最大句子数

编辑 `player.html`：

```javascript
const maxSentencesPerPage = 5;
```

### 修改显示样式

编辑CSS部分的 `.subtitle-line` 类：

```css
.subtitle-line {
    font-size: clamp(20px, 4vw, 32px);  /* 字体大小 */
    font-weight: 700;                    /* 字体粗细 */
    color: #fff;                        /* 字体颜色 */
}
```

## 技术特点

- **流式加载**：按需获取内容，减少内存占用
- **自动预加载**：播放时智能预加载下一段内容
- **章节切换**：支持快速切换不同章节
- **响应式设计**：自动适应不同屏幕尺寸
- **流畅动画**：句子显示带淡入效果
- **智能计算**：根据窗口大小自动调整每页句子数量

## 注意事项

1. 需要先启动分句服务才能使用
2. 确保服务端的 `resource/` 目录包含书籍文件
3. 书籍目录结构要求：`resource/书名/章节.txt`
4. txt文件建议使用UTF-8编码
