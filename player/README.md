# Player - 文本播放器

一个基于Web的文本文件播放器，将txt文件以类似字幕的形式自动播放。

## 功能特性

1. **播放窗口**
   - 占满屏幕宽度
   - 固定1/2屏幕高度
   - 显示当前页的句子

2. **文件读取**
   - 支持选择本地txt文件
   - 自动调用分句API进行句子分割
   - API不可用时自动降级到本地分句

3. **字幕显示**
   - 大字体、加粗显示
   - 类似字幕效果
   - 自动计算最多可显示的句子数
   - 淡入动画效果

4. **自动播放**
   - 支持三种播放速度：慢(3s)、中(2s)、快(1s)
   - 播放/暂停控制
   - 上一页/下一页导航
   - 跳转到开头/结尾

## 使用方法

### 1. 启动分句服务

首先在另一个终端启动分句服务：

```bash
cd ../server
python service_manager.py start
```

### 2. 打开播放器

直接在浏览器中打开 `player.html` 文件：

```
直接双击 player.html 文件
或在浏览器中打开: file:///path/to/player/player.html
```

### 3. 操作步骤

1. 点击「选择文件」按钮，选择一个txt文件
2. 等待分句处理完成
3. 点击绿色播放按钮开始播放
4. 使用速度按钮调整播放速度
5. 使用导航按钮手动翻页

## 界面说明

### 播放窗口（上半部分）
- 显示当前页的句子
- 句子以弹入动画逐个显示
- 背景采用深色渐变，适合长时间观看

### 控制区域（下半部分）

| 控件 | 功能 |
|------|------|
| 进度条 | 显示播放进度，点击可跳转 |
| 速度按钮 | 设置每页停留时间 |
| 选择文件 | 上传txt文件 |
| ⏮ | 跳转到开头 |
| ◀ | 上一页 |
| ▶/⏸ | 播放/暂停 |
| ▶ | 下一页 |
| ⏭ | 跳转到结尾 |

## API配置

默认连接到 `http://localhost:8000` 的分句服务。

如需修改API地址，编辑 `player.html` 中的：

```javascript
const API_URL = 'http://localhost:8000';
```

## 自定义配置

### 修改默认播放速度

编辑 `player.html`：

```javascript
let playSpeed = 2000; // 毫秒，2000 = 2秒
```

### 修改每页最大句子数

```javascript
const MAX_SENTENCES_PAGE = 5;
```

### 修改显示样式

编辑CSS部分的 `.subtitle-line` 类：

```css
.subtitle-line {
    font-size: clamp(20px, 4vw, 32px);  /* 字体大小 */
    font-weight: 700;                    /* 字体粗细 */
    color: #fff;                        /* 字体颜色 */
}
```

## 技术特点

- **响应式设计**：自动适应不同屏幕尺寸
- **自动降级**：API不可用时使用本地分句
- **流畅动画**：句子显示带淡入效果
- **智能计算**：根据窗口大小自动调整每页句子数量
- **键盘友好**：所有操作支持键盘导航

## 注意事项

1. 建议文件大小小于10MB，以保证流畅播放
2. txt文件建议使用UTF-8编码
3. 分句服务需要在后台运行以获得最佳效果
4. 长文本会自动分页，无需手动拆分
